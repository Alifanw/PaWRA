## ğŸ“ Support & Troubleshooting

### Common Issues

**Q: Theme not applying to new widgets?**
A: Use `theme_manager.get_color()` and `theme_manager.get_font()` for all styling.

**Q: Notifications not showing?**
A: Check if previous notification is still visible (4s timeout). Queue is processed sequentially.

**Q: Responsive scaling not working?**
A: Call `responsive.calculate_scale()` after major layout changes.

**Q: Buttons not responding?**
A: Ensure `command=callback` is set correctly, not `command=callback()`.

**Q: Theme not updating for custom widgets?**
A: Implement widget update in `_update_widget_theme()` or manually refresh.
